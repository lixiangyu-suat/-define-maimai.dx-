# 模板题描述

给出项数为 n 的整数数列 a1…n​。

定义函数 f(i) 代表数列中第 i 个元素之后第一个大于 ai​ 的元素的**下标**，即 f(i)=mini<j≤n,aj​>ai​​{j}。若不存在，则 f(i)=0。

试求出 f(1…n)。

        }
