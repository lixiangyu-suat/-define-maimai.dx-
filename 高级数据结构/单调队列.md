# 适用场景
- 维护每一个小区间[i-k+1, i]的最值（0 <= i-k+1, i < n)

| **判断目的**  | **思考方式**            | **公式**                   |
| --------- | ------------------- | ------------------------ |
| **窗口左边界** | 窗口内有 $k$ 个元素，包含 $i$ | $L = i - k + 1$          |
| **队首失效**  | 队首下标小于左边界           | `dq.front() < i - k + 1` |
| **开始输出**  | 当前下标达到窗口长度          | `i >= k`                 |
| **队列比较**  | 永远记得 `dq` 存的是索引     | `a[i] vs a[dq.back()]`   |
[[双队列]]
==细节：队内 存索引值，比较时注意==

| **维度**    | **双队列（Huffman 贪心）**                   | **单调队列（滑动窗口）**                            |
| --------- | ------------------------------------- | ----------------------------------------- |
| **单调性来源** | **外部性**。由贪心策略保证：两个单调递增序列合并出的新序列也一定单调。 | **内部性**。由“剔除逻辑”强制维护：新元素入队前，踢掉所有破坏单调性的老元素。 |
| **操作端点**  | **只在队首 pop**。标准的 FIFO。                | **两端都能 pop**。队尾踢“弱者”，队首踢“过期者”。            |
| **维护目标**  | 维护**全局**的有序性，避开 $O(N \log N)$ 的堆。     | 维护**局部窗口**的最优值（最大/最小）。                    |
| **适用场景**  | 处理所有数据，通常是全局最优解问题。                    | 处理流数据或固定区间问题。                             |

```c可AC
#include <bits/stdc++.h>

using namespace std;

vector<int> a;

int main(){

    #ifndef ONLINE_JUDGE

        freopen("input.txt", "r", stdin);

    #endif


    int n, k; cin >> n >> k;

    a.resize(n+1);

    deque<int> dq;//下标

    for (int i = 1; i <= n; i++){

        cin >> a[i];

    }

    for (int i = 1; i <= n; i++){

        if (!dq.empty() && i-dq.front() >= k) dq.pop_front();

        while (!dq.empty() && a[i] <= a[dq.back()]){//单调递增

            dq.pop_back();

        }

        dq.push_back(i);

        if (i >= k) cout << a[dq.front()] << ' ';

    }

    cout << endl;

    dq.clear();

    for (int i = 1; i <= n; i++){

        if (!dq.empty() && i-dq.front() >= k) dq.pop_front();

        while (!dq.empty() && a[i] >= a[dq.back()]){

            dq.pop_back();

        }

        dq.push_back(i);

        if (i >= k) cout << a[dq.front()] << ' ';

    }

    //窗口左边界： L = i-k+1

    return 0;

}

```
