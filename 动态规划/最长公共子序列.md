# O(nlogn)方法，仅串没有重复元素
- 此时可以把LCS（a,b)转化为求LIS(c);

## 如何求LIS(c)

- 把字符串352978看作一系列牌； (负数序列举例：-7 -5 -8 -1 -3 -2)
- 发到一张牌，如果比所有牌堆上最小的牌都大（或没有牌堆），那么新开一个堆；
- 如果比部分牌堆上的【最小牌】（可以理解为牌堆最顶上的牌）小，那么放在第一个堆（贪心二分思想，logN）
- 例如：3-> 3,5->2,5->2,5,9->2,5,7->2,5,7,8。则`LIS(c) = len() = 4`

```c
// 针对排列的 O(n log n) 模板
// 假设 A 和 B 长度均为 n
int pos[N], C[N], low[N];

// 1. 建立映射
for (int i = 1; i <= n; i++) pos[A[i]] = i;
// 2. 转化序列
for (int i = 1; i <= n; i++) C[i] = pos[B[i]];

// 3. 求 C 的 LIS (贪心 + 二分)
int len = 0;
    for (auto c : s){ //s是一个序列
        int id = lower_bound(low, low+len, c) - low; //low是数组第一个下标的地址
        //upper_bound为第一个大于i的下标地址
        low[id] = c;
        if (id == len) len++;
    }
// 最终 len 即为 LCS 长度
```

# O(n^2)方法

想象两个字符串 $A$ = `ABCBDAB` 和 $B$ = `BDCABA`。

1. **相等时**：如果当前字符匹配，说明找到了公共子序列的一个新成员，长度在去掉这两个字符后的基础上 $+1$（即取左上方格子 $+1$）。
    
2. **不等时**：说明当前两个字符无法同时作为结尾，长度取决于“舍弃 $A$ 的当前字符”或“舍弃 $B$ 的当前字符”中更好的结果（即取左方或上方格子的最大值）。
    

### 伪代码

```c
FUNCTION Standard_LCS(A, B):
    n = length(A)
    m = length(B)
    CREATE 2D_ARRAY dp[n+1][m+1] INITIALIZED TO 0
    
    FOR i FROM 1 TO n:
        FOR j FROM 1 TO m:
            IF A[i] == B[j]:
                # 发现匹配，斜向对角线加 1 //可见只依赖相邻格子，空间复杂度可优化为O(N)
                dp[i][j] = dp[i-1][j-1] + 1
            ELSE:
                # 不匹配，取左边或上边的最大值
                dp[i][j] = MAX(dp[i-1][j], dp[i][j-1])
                
    RETURN dp[n][m] # 右下角即为答案
```

# 位运算方法，O(n*m/w)

这里把`dp[i][j]`理解为：串A的前j长度和串B的前i长度的最长公共子序列为`dp[i][j]`
`p[b[i]][j] = 1意味着原来的a[j] == b[i]`

- $X = M_{i-1} \mid p(B_i)$
我们直接看 $p(B_i)$。如果第 $j$ 位是 $1$，就等同于那个 `if` 条件成立了。
**$M_{i-1}$** 告诉我们：上一行在哪里已经有“台阶”了。
$M_{0}$ 指代：0000000000(未匹配)
那么X代表 “旧状态$M_{i-1}$”与“新机会$p(B_i)$”的合集，代表了当前行所有“可能成为 LCS 贡献点”的位置。
- $V = (M_{i-1} \ll 1) \mid 1$

## 实现步骤

### 1. 基础差分定义

定义 $f_{i,j}$ 为标准 DP 状态，$M_{i,j}$ 为位运算优化的差分状态：

$$M_{i,j} = f_{i,j} - f_{i,j-1}$$

其中 $M_{i,j} \in \{0, 1\}$。

### 2. 字符位置掩码 (Precomputation)

定义字符集为 $\Sigma$，对于每一个字符 $c \in \Sigma$，预处理一个长度为 $|A|$ 的位向量 $p(c)$：

$$p(c)_j = \begin{cases} 1 & \text{if } A_j = c \\ 0 & \text{if } A_j \neq c \end{cases}$$

### 3. 核心行转移公式 (Core Transition)

在处理字符串 $B$ 的第 $i$ 个字符时，从上一行状态 $M_{i-1}$ 转移到当前行 $M_i$：

**第一步：构造候选位 $X$**

$$X = M_{i-1} \mid p(B_i)$$

**第二步：构造分段阻断位 $V$**

$$V = (M_{i-1} \ll 1) \mid 1$$

**第三步：计算当前行差分向量 $M_i$**

$$M_i = (X \oplus (X - V)) \& X$$

> **注意**：在实际代码实现中，位移方向和补位方向（左移还是右移）取决于你如何定义下标 $j$ 在二进制位中的映射关系。

### 4. 最终答案提取 (Final Result)

最长公共子序列的长度等于最后一行 $M$ 中 $1$ 的总个数：

$$LCS(A, B) = \text{popcount}(M_{|B|}) = \sum_{j=1}^{|A|} M_{|B|, j}$$
