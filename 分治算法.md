# 🚀 分治算法 (Divide and Conquer) 快速手册

## 1. 核心思想
- **Divide (分)**: 将原问题划分为子问题。
- **Conquer (治)**: 递归解决子问题。若子问题足够小，直接求解。
- **Combine (合)**: 将子问题的解合并为原问题的解。

## 2. 经典应用场景
| 领域        | 典型算法             | 时间复杂度                                 |
| :-------- | :--------------- | :------------------------------------ |
| **排序**    | 归并排序 / 快速排序      | $O(n \log n)$                         |
| **搜索**    | 二分查找             | $O(\log n)$                           |
| **数学**    | Karatsuba 高精度乘法  | $O(n^{\log_2 3}) \approx O(n^{1.58})$ |
| **数学**    | FFT (快速傅里叶变换)    | $O(n \log n)$                         |
| **DP 优化** | 决策单调性分治(如泛化物品叠加) | $O(C \log C)$                         |
| **几何**    | 最近点对问题           | $O(n \log n)$                         |
| **树论**    | 点分治 / 边分治        | $O(n \log n)$                         |

## 3. DP 分治优化模板 (决策单调性)
当 $f[i][j] = \min_{k < j} \{ f[i-1][k] + w(k, j) \}$ 且最优决策点 $opt(j)$ 随 $j$ 单调递增时适用：

```cpp
// L, R: 当前处理的状态区间；optL, optR: 对应的决策点搜索区间
void solve(int L, int R, int optL, int optR) {
    if (L > R) return;
    int mid = (L + R) >> 1, bestK = optL;
    
    // 1. 暴力找 mid 的最优决策点
    for (int k = optL; k <= min(mid, optR); k++) {
        if (calc(k, mid) > val(bestK, mid)) {
            bestK = k;
        }
    }
    
    ans[mid] = val(bestK, mid);
    
    // 2. 根据决策单调性缩小范围递归
    solve(L, mid - 1, optL, bestK);
    solve(mid + 1, R, bestK, optR);
}
```
## 4. 判别技巧

- 子问题是否**相互独立**？（独立则分治，不独立且有重叠子问题则 DP）
    
- 决策点是否具有**单调性**？（是则 $O(C^2) \to O(C \log C)$）
