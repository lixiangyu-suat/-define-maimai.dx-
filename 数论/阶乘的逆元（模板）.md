对于大数据的组合数学问题，**预处理阶乘及其逆元**是最简单、标准且高效的方法。

当模数 $P$ 是质数时，
根据费马小定理：$a^{P-1} \equiv 1 \pmod P$。 
变形得：$a \cdot a^{P-2} \equiv 1 \pmod P$。 
所以：**$a$ 的逆元就是 $a^{P-2} \pmod P$**。

---
### 1. 更简单的方法：预处理

不要在循环里反复调用 `modInverse`（它是 $O(\log MOD)$），而是先花 $O(N)$ 时间把所有阶乘和逆元存进数组，后面直接 $O(1)$ 取用。

**核心代码模板：**
这个循环会一直运行到 `i = 0`。最终会算出 `invFact[0]`。如果你能正确算出 `invFact[0]` 且结果为 $1$，说明你的整个递推链条是完美的。
```c
ll fact[500005], invFact[500005];

ll power(ll a, ll b) {
    ll res = 1;
    for (a %= MOD; b; b >>= 1, a = a * a % MOD) //快速幂，a为底，res为目标数; O(log MOD)
        if (b & 1) res = res * a % MOD;
    return res;
}

void initfactory(int N) {
    fact[0] = 1;
    for (int i = 1; i <= N; i++) fact[i] = fact[i - 1] * i % MOD;
    invFact[N] = power(fact[N], MOD - 2); 
    for (int i = N - 1; i >= 0; i--) invFact[i] = invFact[i + 1] * (i + 1) % MOD;// 阶乘逆元的递推公式：inv(n!) = inv((n+1)!) * (n+1)
    //这里的invfact[1]都可能是一个很大的数，所有全程使用ll
}
```

---
### 2. 如何使用？

- **求 $n!$：** 直接用 `fact[n]`。
- **求 $\frac{1}{n!}$：** 直接用 `invFact[n]`。
- **计算多重排列 $\frac{(N-2)!}{\prod c_i!}$：** `ll ans = fact[N-2];`
	`for(int cnt : counts) ans = ans * invFact[cnt] % MOD;`

---

### 3. 注意点（避坑指南）


1. **模数必须为质数**：费马小定理（`power(a, MOD-2)`）仅在 $MOD$ 为质数（如 $10^9+7$, $998244353$）时成立。
2. **需要在输入n后手动初始化**: `initfactory(n)`
3. **数组大小**：预处理的 $N$ 必须覆盖题目给出的最大范围。
4. **LL 强转**：任何两个可能超过 $10^9$ 的数相乘，必须先转 `long long`，即 `(ll)a * b % MOD`。
5. **0 的阶乘**：务必记得 `fact[0] = 1` 和 `invFact[0] = 1`。
---
### 4. 以后涉及组合数（$C_n^k$）


$$C_n^k \equiv n! \cdot (k!)^{-1} \cdot ((n-k)!)^{-1} \pmod P$$
**代码表达：**
```
ll C(int n, int k) {
    if (k < 0 || k > n) return 0;
    return fact[n] * invFact[k] % MOD * invFact[n - k] % MOD;
}
```

这种预处理方法是算法竞赛的**标准配置**，既能防精度损失，又能极大提高运行速度。

```c
#include <bits/stdc++.h>

using namespace std;

using ll = long long;

  

const int N = 500005, M = 1e9+7;

ll fact[N], invfact[N];

  

ll power(ll a, ll n){//这里参数等也要传很多 ll

    ll ans = 1;

    while (n){

        if (n&1) ans = ans * a % M; //乘以底数a

        a = a * a % M; //底数自乘

        n >>= 1;

    }

    return ans;

}

  

void initfact(int n){

    fact[0] = 1;

    for (int i = 1; i <= n; i++){

        fact[i] = fact[i-1] * i % M;

    }

    invfact[n] = power(fact[n], M-2);

    for (int i = n-1; i >= 0; i--){

        invfact[i] = invfact[i+1] * (i+1) % M;

    }

}

  

int main(){

    initfact(500000); //记得声明初始化

    cout << fact[1] << ' ' << invfact[1] << endl;

    while (true){

        int a, b;

        cin >> a >> b;

        if (a < b) {

            cout << "a < b!!!" << endl;

            continue;

        }

        ll ans = 1;

        ans = ans * fact[a] % M;

        ans = ans * invfact[b] % M;

        ans = ans * invfact[a-b] % M;

        printf("C(%d, %d) = %d\n", a, b, ans);

    }

    return 0;

}
```
